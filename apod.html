Astronomic Picture Of the Day

🚀

Bem-vindo ao projeto de aplicação da API APOD, você deve perguntar-se o que é isso? Certo vamos lá.
O Astronomy Picture of the Day (APOD, em português: Imagem Astronômica do Dia) é um website mantido pela NASA e pela Universidade Tecnológica de Michigan. De acordo com o website, "cada dia uma imagem ou foto diferente do universo é mostrada, com uma curta explicação escrita por um astrônomo profissional".Basicamente utilizando a API da NASA foi desenvolvida uma página que quando for inserida uma data a partir de 16 de junho de 1995 uma imagem diferente será mostrada com sua respectiva foto, sendo assim divirta-se nessa viagem e explore o universo pelos olhos da NASA.
